# Utilities

Contains the following utilities:

### Custom Installs
kubectl
terraform
vault
kops
helm

### APK Packages
curl
postgresql-client
git
net-tools
nmap
tcpdump
iputils
bind-tools
mysql-client
vim
jq
bash
bash-completion
iptables
openssh-client
openssl

## Kubernetes

There is a simple pod definition yaml file that will add this to your kube cluster for diagnosing issues inside the cluster
